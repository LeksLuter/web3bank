# Web3 Bank

Your Decentralized Crypto-Mem Adventure!

## Description

Web3 Bank is a decentralized web application that provides a fun and engaging interface for crypto enthusiasts. Built with modern web technologies, it offers a unique blend of banking concepts with meme culture.

## Features

- **Multi-language Support**: Switch between English and Russian
- **Wallet Integration**: Connect with Fantom Wallet
- **Responsive Design**: Optimized for both desktop and mobile devices
- **Interactive UI**: Modern animations and user-friendly interface
- **Quest Room**: Interactive quest experience
- **Admin Panel**: Administrative controls

## Project Structure

```
├── index.html          # Main landing page
├── quest-room.html     # Quest room page
├── admin.html          # Admin panel
├── styles.css          # Stylesheet
├── scripts.js          # JavaScript functionality
├── favicon.ico         # Site icon
└── img/                # Images directory
```

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Google Fonts (Poppins)
- Fantom Wallet Integration

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. No build process required - it's a static website!

## Pages

- **Home** (`index.html`): Main landing page with navigation and features
- **Quest Room** (`quest-room.html`): Interactive quest experience
- **Admin** (`admin.html`): Administrative dashboard
